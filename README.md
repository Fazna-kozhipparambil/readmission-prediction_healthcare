#  Hospital Readmission Prediction using Machine Learning

##  Project Overview
This project predicts whether a patient will be readmitted to the hospital based on medical and hospital-related features. The goal is to help healthcare providers identify high-risk patients and reduce readmission rates.

##  Objectives
- Predict hospital readmission (Yes/No)
- Identify key factors influencing readmission
- Compare multiple machine learning models
- Improve model performance using evaluation metrics

##  Dataset
The dataset contains patient demographic and hospital stay information.  
Target variable: `readmitted` (0 = No, 1 = Yes)

##  Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- XGBoost
- Jupyter Notebook / Google Colab

##  Models Used
- Logistic Regression  
- Decision Tree  
- Random Forest  
- XGBoost (Best Performing Model)

## Model Performance
- Accuracy: ~62%  
- Precision: 0.60  
- Recall: 0.50  
- F1-score: 0.55
